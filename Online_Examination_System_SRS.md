# Software Requirements Specification (SRS)

# Online Examination Management System

## 1. Introduction

### 1.1 Purpose
The purpose of this project is to develop a web-based Online Examination Management System that allows educational institutions to conduct examinations digitally.

### 1.2 Scope
The system will provide:
- Student registration and authentication
- Online examination conduction
- Question bank management
- Automatic evaluation
- Result generation
- Administrative control

## 2. Functional Requirements

### Student Module
- Registration/Login
- View available exams
- Attend exams
- Submit answers
- View results

### Faculty/Admin Module
- Create exams
- Manage question bank
- Schedule exams
- Assign exams
- Publish results

### Examination Features
- MCQ Questions
- Timer functionality
- Auto submission
- Random question order
- Negative marking (optional)

### Reports
- Result generation
- Student performance analysis
- Attendance reports

## 3. Non-Functional Requirements

- Secure login
- High availability
- Data backup
- Responsive design
- Scalability
- Reliability
